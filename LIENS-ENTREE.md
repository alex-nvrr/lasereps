# Liens d’entrée LaserEPS (GitHub Pages)

**URL de base :** [https://alex-nvrr.github.io/lasereps/](https://alex-nvrr.github.io/lasereps/)

| Usage | Lien à partager (QR, mail, ENT…) |
|--------|----------------------------------|
| **Élève** — parcours classe, sans bouton « Changer de mode » après inscription | `https://alex-nvrr.github.io/lasereps/?entry=eleve` |
| **Prof** — création de classes, code `PROF` la première fois | `https://alex-nvrr.github.io/lasereps/?entry=prof` |
| **Libre / perso** — stats personnelles sans classe | `https://alex-nvrr.github.io/lasereps/?entry=libre` |

Alias acceptés : `?entry=perso` ou `?entry=personnel` (équivalent à libre).

Le paramètre `entry` est mémorisé pour la session du navigateur. Une fois connecté, la barre d’adresse est **synchronisée** avec le mode (`?entry=prof`, `?entry=eleve` ou `?entry=libre` selon le cas). À la déconnexion, `entry` est retiré de l’URL.
